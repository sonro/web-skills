HTML Tags

This page will outline the uses of certain HTML tags

Basic HTML

<!DOCTYPE>
Defines the document type. For HTML5 simply use <!DOCTYPE html>.
<html>
Defines the contents of the HTML document. Becomes the root of the document. All other nodes/tags should be contained within (except <!DOCTYPE>). Will often contain a language attribute to assist search engines, screen readers and potentially browsers in cataloging and displaying the page. For example: <html lang="en">.
